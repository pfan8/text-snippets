# Text Snippets

A simple repository for storing and syncing text snippets across multiple devices.

## Usage

Store your text snippets in this repository and sync them across your devices using git.

## Structure

You can organize your snippets however you like:
- Create separate `.txt` or `.md` files for different categories
- Use folders to organize by topic or device
- Add timestamps or tags to your snippets

## Quick Start

1. Clone this repository on any device
2. Add your text snippets
3. Commit and push changes
4. Pull on other devices to sync
